## 🚀 Installation

### Step 1: Download the Extension
1. Download all extension files to a folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the extension folder

### Step 2: Configure API Keys
1. Click the extension icon in Chrome toolbar
2. Enter your OpenAI API key (get from [OpenAI Platform](https://platform.openai.com/api-keys))
3. Click "Connect Google Account" to authenticate
4. Test the connection

### Step 3: Start Using!
1. Go to Gmail
2. Look for the "🤖 AI Assistant" button in the top right
3. Click to open your personal email assistant
4. Start chatting!
